# Atlas Eval Candidates Review Checklist

Use this for weekly review of `atlas-eval-candidates.json`.

- [ ] Is this a real failure pattern, not a one-off weird case?
- [ ] Could this failure cause financial harm, confusion, or trust loss?
- [ ] Is the candidate eval clearly written and realistic?
- [ ] Does it test a genuinely new gap instead of duplicating an existing case?
- [ ] Decision: **Approve / Revise / Reject**
