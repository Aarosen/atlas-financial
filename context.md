Deterministic Tier Assignment Rules (v1)
Use the config as a snapshot.
Tier logic (deterministic order)
	1	Foundation if ANY is true:
	•	buffer_months < tiers.foundation.buffer_months_max
	•	future_alloc_pct <= tiers.foundation.future_alloc_pct_max
	•	OR debt_exposure_level == Critical (from debt module)
	2	Else Stabilizing if within stabilizing ranges (buffer and future allocation) OR if debt exposure is High.
	3	Else Strategic if within strategic ranges and debt exposure ≤ Moderate.
	4	Else Growth-Ready if buffer + future allocation meet growth-ready minimums and no high-interest debt severity.
Important: Debt exposure can “pull someone down” a tier, but never “push up” a tier.
This preserves safety.

Config Validation Rules (must pass before use)
On fetch, validate:
Required
	•	version is non-empty string
	•	ttl_seconds between 60 and 3600
	•	All tier fields exist
Numeric sanity
	•	All mins/maxes are >= 0
	•	For each tier: min <= max when both exist
	•	Tier ranges must be contiguous and non-overlapping:
	◦	stabilizing.buffer_min == foundation.buffer_max
	◦	strategic.buffer_min == stabilizing.buffer_max
	◦	growth_ready.buffer_min == strategic.buffer_max
Guardrails
	•	max_future_alloc_pct <= 0.8 (hard safety)
	•	Debt ratio thresholds must be increasing:
	◦	moderate_min < high_min < critical_min
If any validation fails → reject config.

Last-Known-Good (LKG) Fallback Behavior
This is the operational safety net.
Engine maintains:
	•	activeConfig (current in-memory)
	•	lkgConfig (persisted copy, e.g., local disk/DB or durable cache)
	•	lkgMetadata (version, fetchedAt, checksum)
Fetch policy
	•	Fetch at startup + refresh every ttl_seconds
	•	On refresh:
	1	Fetch new config
	2	Validate schema + sanity
	3	If valid → promote to activeConfig and update lkgConfig
	4	If invalid → keep using lkgConfig, raise alert
If provider is down
	•	Continue using lkgConfig
	•	If no LKG exists (first boot):
	◦	Use a bundled defaultConfig shipped with the engine
	◦	Mark outputs with config_source: "bundled_default"
	◦	Trigger high-priority alert
Output must always include:
	•	config_version_used
	•	config_source = managed|lkg|bundled_default
	•	config_fetched_at
This guarantees you can always explain results and reproduce them.

Where this plugs into your engine
	•	Only the orchestrator holds the config snapshot.
	•	Services receive (input, configSnapshot) as arguments.
	•	Services never fetch config.
This keeps everything deterministic and testable.

import type { IDebtExposureService } from "../services/baselineStrategy/debtExposure.service";
import type { IStabilityTierService } from "../services/baselineStrategy/stabilityTier.service";
import type { IUpgradeLeverService } from "../services/baselineStrategy/upgradeLever.service";

export interface BaselineStrategyOrchestratorDeps {
  bufferService: IBufferService;
  allocationService: IAllocationService;
  debtExposureService: IDebtExposureService;
  stabilityTierService: IStabilityTierService;
  upgradeLeverService: IUpgradeLeverService;
}

export interface BaselineStrategyOrchestratorMeta {
  configVersionUsed: string;
  configSource: ConfigSource;
  configFetchedAt: string; // ISO
}

export class BaselineStrategyOrchestrator {
  constructor(private readonly deps: BaselineStrategyOrchestratorDeps) {}

  run(input: FinancialInputV1, config: AtlasStrategyConfigV1, meta: BaselineStrategyOrchestratorMeta): BaselineStrategyOutputV1 {
    const buffer = this.deps.bufferService.compute(input, config);
    const alloc = this.deps.allocationService.compute(input, config);
    const debt = this.deps.debtExposureService.compute(input, config);

    const tier = this.deps.stabilityTierService.compute(
      {
        bufferMonths: buffer.bufferMonths,
        futureAllocationPercent: alloc.futureAllocationPercent,
        debtExposureLevel: debt.debtExposureLevel,
      },
      config
    );

    const levers = this.deps.upgradeLeverService.compute(
      {
        stabilityTier: tier,
        bufferMonths: buffer.bufferMonths,
        futureAllocationPercent: alloc.futureAllocationPercent,
        debtExposureLevel: debt.debtExposureLevel,
        isNegativeCashflow: alloc.futureAllocationPercent < 0,
      },
      config
    );

    return {
      stabilityTier: tier,
      bufferMonths: buffer.bufferMonths,
      futureAllocationPercent: alloc.futureAllocationPercent,
      debtExposureLevel: debt.debtExposureLevel,
      primaryUpgradeLever: levers.primaryUpgradeLever,
      secondaryUpgradeLever: levers.secondaryUpgradeLever,
      configVersionUsed: meta.configVersionUsed,
      configSource: meta.configSource,
      configFetchedAt: meta.configFetchedAt,
      assumptionsUsed: {
        discretionaryDerived: alloc.discretionaryDerived,
        futureAllocCapped: alloc.futureAllocCapped,
      },
    };
  }
}

  // 1) Hard rule: negative cashflow
  if input.isNegativeCashflow and enabled("stabilize_cashflow"):
    return { primary: "stabilize_cashflow", secondary: enabled("optimize_discretionary_spend") ? "optimize_discretionary_spend" : undefined }

  // 2) Hard rule: high debt exposure
  if (input.debtExposureLevel == "High" || input.debtExposureLevel == "Critical") and enabled("eliminate_high_interest_debt"):
    // secondary preference: stabilize_cashflow if future alloc is low, else buffer
    secondary =
      (input.futureAllocationPercent < 0 && enabled("stabilize_cashflow")) ? "stabilize_cashflow"
      : enabled("build_emergency_buffer") ? "build_emergency_buffer"
      : undefined

  // compute next tier targets
  next = nextTier(input.stabilityTier)
  targetBuffer = config.targets.emergency_buffer_months_target_by_tier[next]
  targetFuture = config.targets.future_alloc_pct_target_by_tier[next]

  // 3) Buffer lever

    secondary = (input.futureAllocationPercent < targetFuture && enabled("increase_future_allocation")) ? "increase_future_allocation" : undefined

  // 4) Future allocation lever

    secondary = (input.bufferMonths < targetBuffer && enabled("build_emergency_buffer")) ? "build_emergency_buffer" : undefined
    return { primary: "increase_future_allocation", secondary }

  // 5) Optimization fallback
  if enabled("optimize_discretionary_spend"):
    return { primary: "optimize_discretionary_spend" }

  // fallback safety (shouldn't happen if config is sane)
  return { primary: "build_emergency_buffer" }

function fmtPct(x?: number): string | undefined {
  if (x === undefined || x === null) return undefined;

function fmtNum(x?: number, decimals = 1): string | undefined {
  if (x === undefined || x === null) return undefined;
  const p = Math.pow(10, decimals);
  return String(Math.round(x * p) / p);
}

import type { ExplainabilityBlock } from "../contracts/explainability";
import type { TemplateStore } from "../contracts/templateStore";
import type { RenderedExplanation, RenderedMessage } from "../contracts/renderedExplanation";

function pickTemplate(store: TemplateStore, candidates: string[]): { text: string; usedKey: string } {
  for (const k of candidates) {
    const t = store.get(k);
    if (t) return { text: t, usedKey: k };
  }
  return { text: "", usedKey: candidates[0] };
}

function substitute(text: string, metrics: any): string {
  const map: Record<string, string | undefined> = {
    bufferMonths: fmtNum(metrics.bufferMonths, 1),
    bufferTargetMonths: fmtNum(metrics.bufferTargetMonths, 1),
    futureAllocPct: fmtPct(metrics.futureAllocPct),
    futureTargetPct: fmtPct(metrics.futureTargetPct),
    hiDebtIncomeRatio: fmtNum(metrics.hiDebtIncomeRatio, 2),
    debtPaymentIncomeRatio: fmtNum(metrics.debtPaymentIncomeRatio, 2),
  };

  return text.replace(/\{(\w+)\}/g, (_, key) => map[key] ?? `{${key}}`);
}

export function renderExplanation(
  explain: ExplainabilityBlock,
  store: TemplateStore
): RenderedExplanation {
  const tone = toneFromUrgency(explain.urgency);

  const keysUsed: string[] = [];

  // --- Message 1: Tier reveal
  const tierKeyBase =
    explain.stabilityTier === "GrowthReady" ? "tier.growth_ready"
    : explain.stabilityTier === "Strategic" ? "tier.strategic"
    : explain.stabilityTier === "Stabilizing" ? "tier.stabilizing"
    : "tier.foundation";

  const reveal = pickTemplate(store, [`${tierKeyBase}.reveal`]);
  const bridge = pickTemplate(store, [`${tierKeyBase}.bridge`]);
  keysUsed.push(reveal.usedKey, bridge.usedKey);

  const msg1: RenderedMessage = {
    role: "assistant",
    tone,
    body: `${reveal.text}\n\n${bridge.text}`
  };

  // --- Message 2: Lever why + next
  const leverKeyBase = explain.templateKeyWhy; // e.g. "lever.build_emergency_buffer"

  const why = pickTemplate(store, [
    `${leverKeyBase}.why.${tone}`,
    `${leverKeyBase}.why.advisory`,
    `${leverKeyBase}.why.calm`
  ]);
  const next = pickTemplate(store, [
    `${leverKeyBase}.next.${tone}`,
    `${leverKeyBase}.next.advisory`,
    `${leverKeyBase}.next.calm`
  ]);
  keysUsed.push(why.usedKey, next.usedKey);

  const whyText = substitute(why.text, explain.metrics);
  const nextText = substitute(next.text, explain.metrics);

  const msg2: RenderedMessage = {
    role: "assistant",
    tone,
    title: "Here’s why",
    body: `${whyText}\n\n${nextText}`
  };

  // --- Message 3: CTA
  const action = explain.nextAction;
  const actionTitle = store.get(`action.${action.actionKey}.title`) ?? action.actionTitle;
  const actionPrompt = store.get(`action.${action.actionKey}.prompt`) ?? action.actionPrompt;
  keysUsed.push(`action.${action.actionKey}.title`, `action.${action.actionKey}.prompt`);

  const noJudgment = store.get("meta.noJudgmentLine") ?? "You’re in control — I’m here to help you think clearly.";

  const payload: Record<string, any> = {
    ...action,
    metrics: explain.metrics,
    lever: explain.primaryLever,
    tier: explain.stabilityTier
  };

  const msg3: RenderedMessage = {
    role: "assistant",
    tone,
    body: actionPrompt,
    ctas: [
      {
        id: `cta.${action.actionKey}.primary`,
        label: actionTitle,
        actionKey: action.actionKey,
        payload,
        variant: "primary"
      },
      {
        id: `cta.ask_question.secondary`,
        label: "Ask Atlas a question",
        actionKey: "ask_question",
        payload: { context: "baseline_explainability" },
        variant: "secondary"
      }
    ],
    footnote: noJudgment
  };

  return {
    messages: [msg1, msg2, msg3],
    telemetry: { urgency: explain.urgency, templateKeysUsed: keysUsed }
  };
}

    if (t) obs.push(substitute(t, m));
  }

  if (m.bufferMonths !== undefined && m.bufferTargetMonths !== undefined && m.bufferMonths < m.bufferTargetMonths) {
    const t = store.get("tracePanel.observations.buffer_low");
    if (t) obs.push(substitute(t, m));
  }

  if (m.futureAllocPct !== undefined && m.futureTargetPct !== undefined && m.futureAllocPct < m.futureTargetPct) {
    const t = store.get("tracePanel.observations.future_low");
    if (t) obs.push(substitute(t, m));
  }

  // Reason by lever
  const reasonKey = `tracePanel.reason.${explain.primaryLever}`;
  const reason = store.get(reasonKey) ?? "This recommendation is the highest leverage next step.";

  const control = store.get("tracePanel.control.default") ??
    "Atlas recommends — you decide. You can always ask questions or adjust the plan.";

  // telemetry: which trace steps were present (optional)
  const usedTraceSteps = trace?.steps?.map(s => s.stepKey) ?? [];

  return {
    tone,
    title,
    sections: [
      { heading: inputsHeading, bullets: inputsBullets },
      { heading: obsHeading, bullets: obs.length ? obs : ["Atlas is using your snapshot to form a stable baseline."] },
      { heading: reasonHeading, body: reason },
      { heading: controlHeading, body: control }
    ],
    footnote: "If anything feels off, tell me — we’ll adjust together.",
    telemetry: { usedTraceSteps }
  };
}

State transition logic (minimal reducer)
/src/onboarding/onboardingReducer.ts

import type { OnboardingState } from "./onboardingState";
import type { OnboardingEvent } from "./onboardingEvents";
import { computeMissingFields } from "./missingFields";

export function onboardingReducer(state: OnboardingState, event: OnboardingEvent): OnboardingState {
  switch (event.type) {
    case "RESET":
      return {
        step: "start",
        ctx: { draft: {}, intent: {}, missingFields: [] },
      };

    case "SET_INTENT": {
      const nextDraft = {
        ...state.ctx.draft,
        ...(event.primaryGoal ? { primaryGoal: event.primaryGoal } : {}),
        ...(event.timeHorizonYears ? { timeHorizonYears: event.timeHorizonYears } : {}),
      };
      const missingFields = computeMissingFields(nextDraft);
      return {
        step: state.step === "start" ? "intent" : state.step,
        ctx: { ...state.ctx, draft: nextDraft, intent: { ...state.ctx.intent, ...event }, missingFields },
      };
    }

    case "EXTRACTED_FIELDS": {
      const nextDraft = { ...state.ctx.draft, ...event.fields };
      const missingFields = computeMissingFields(nextDraft);
      return {
        step: state.step,
        ctx: { ...state.ctx, draft: nextDraft, missingFields, extractionConfidence: event.confidence },
      };
    }

    case "CONFIRM_FIELDS": {
      const nextDraft = { ...state.ctx.draft, ...event.confirmed };
      const missingFields = computeMissingFields(nextDraft);
      return { ...state, ctx: { ...state.ctx, draft: nextDraft, missingFields } };
    }

    case "GENERATE_BASELINE":
      return { ...state, step: "review" };

    case "BASELINE_READY":
      return { ...state, step: "baseline_generated" };

    default:
      return state;
  }
}

Question selection policy (the “guided but adaptive” part)

import type { OnboardingContext } from "./onboardingState";

export type QuestionKey =
  | "ask_goal"
  | "ask_time_horizon"
  | "ask_income"
  | "ask_essentials"
  | "ask_savings"
  | "ask_hi_debt"
  | "ask_low_debt"
  | "ask_debt_payments"
  | "ask_risk_tolerance";

export function chooseNextQuestion(ctx: OnboardingContext): QuestionKey {
  // 1) Ensure intent exists early (emotion-first)
  if (!ctx.intent.primaryGoal) return "ask_goal";
  if (!ctx.intent.timeHorizonYears) return "ask_time_horizon";

  // 2) Then gather baseline numbers in a safe order
  if (ctx.missingFields.includes("monthlyIncome")) return "ask_income";
  if (ctx.missingFields.includes("essentialExpenses")) return "ask_essentials";
  if (ctx.missingFields.includes("totalSavings")) return "ask_savings";

  // Debt is sensitive; ask after some trust is formed
  if (ctx.missingFields.includes("highInterestDebt")) return "ask_hi_debt";
  if (ctx.missingFields.includes("lowInterestDebt")) return "ask_low_debt";
  if (ctx.missingFields.includes("monthlyDebtPayments")) return "ask_debt_payments";

  if (ctx.missingFields.includes("riskTolerance")) return "ask_risk_tolerance";

  // If nothing missing, proceed
  return "ask_risk_tolerance";
}

import { onboardingReducer } from "./onboardingReducer";
import { computeMissingFields } from "./missingFields";
import { chooseNextQuestion } from "./nextQuestion";
import { uiHintFor } from "./questionUiHints";
import { finalizeFinancialInput } from "./finalizeFinancialInput";
import { renderExplanation } from "../renderers/atlasStrategyExplanationRenderer";
import { renderTracePanel } from "../renderers/tracePanelRenderer";

export class OnboardingController {
  constructor(private deps: any) {}

  async handleUserMessage(state: any, userText: string, locale: string) {
    // 1) Extract
    const extraction = await this.deps.extractionClient.extract(userText, {
      locale,
      prior: state.ctx?.draft ?? {},
    });

    // 2) Reduce: fields
    let nextState = onboardingReducer(state, {
      type: "EXTRACTED_FIELDS",
      fields: extraction.fields,
      confidence: extraction.confidence,
    });

    // 3) Reduce: intent (if present)
    if (extraction.intent?.primaryGoal || extraction.intent?.timeHorizonYears || extraction.intent?.biggestConcern) {
      nextState = onboardingReducer(nextState, {
        type: "SET_INTENT",
        primaryGoal: extraction.intent.primaryGoal,
        timeHorizonYears: extraction.intent.timeHorizonYears,
        biggestConcern: extraction.intent.biggestConcern,
      });
    }

    // 4) Ensure missingFields is fresh
    const missing = computeMissingFields(nextState.ctx.draft);
    nextState = { ...nextState, ctx: { ...nextState.ctx, missingFields: missing } };

    // 5) If incomplete -> ask next question
    if (missing.length > 0) {
      const qKey = chooseNextQuestion(nextState.ctx);
      const qText = this.deps.templateStore.get(`q.${qKey}`) ?? "A quick question to sharpen this…";
      const ui = uiHintFor(qKey);

      return {
        state: nextState,
        messages: [
          { role: "user", body: userText },
          { role: "assistant", tone: "calm", body: qText }
        ],
        expectedInput: { questionKey: qKey, ...ui },
      };
    }

    // 6) Complete -> run baseline
    const finalized = finalizeFinancialInput(nextState.ctx.draft);

    const { config, meta } = await this.deps.configProvider.getStrategyConfig();
    const baseline = await this.deps.engineClient.runBaseline(finalized, config, meta);

    // 7) Render explanation
    const renderedExplanation = renderExplanation(baseline.explainability, this.deps.templateStore);

    // 8) Render trace panel (optional)
    const panel = renderTracePanel(
      baseline.explainability,
      this.deps.templateStore,
      baseline.explainability.decisionTrace
    );

    // 9) Attach panel to Message 2 (“Here’s why”) if present
    const msgs = renderedExplanation.messages.map((m, i) => ({
      role: "assistant",
      tone: m.tone,
      title: m.title,
      body: m.body,
      ctas: m.ctas,
      footnote: m.footnote,
      expandablePanel: i === 1 ? panel : undefined
    }));

    return {

export function fmtPct(x?: number): string | undefined {
  if (x === undefined || x === null || !isFinite(x)) return undefined;
  return String(Math.round(x * 100));
}

export function fmtNum(x?: number, decimals = 1): string | undefined {
  if (x === undefined || x === null || !isFinite(x)) return undefined;
  const p = Math.pow(10, decimals);
  return String(Math.round(x * p) / p);
}

import fs from "fs";
import path from "path";
import type { TemplateStore } from "../contracts/templateStore";

export class JsonTemplateStore implements TemplateStore {
  private map: Map<string, string> = new Map();

  constructor(private readonly rootDir: string, private readonly locale: string) {
    this.load();
  }

  private load() {
    const dir = path.join(this.rootDir, "templates", this.locale);
    const files = fs.readdirSync(dir).filter((f) => f.endsWith(".json"));

    for (const file of files) {
      const obj = JSON.parse(fs.readFileSync(path.join(dir, file), "utf-8"));
      this.flatten(obj);
    }
  }

  private flatten(obj: any, prefix = "") {
    for (const [k, v] of Object.entries(obj)) {
      const key = prefix ? `${prefix}.${k}` : k;
      if (typeof v === "string") this.map.set(key, v);
      else if (typeof v === "object" && v !== null) this.flatten(v, key);
    }
  }

  get(pathKey: string): string | undefined {
    return this.map.get(pathKey);
  }
}

export function onboardingReducer(state: OnboardingState, event: OnboardingEvent): OnboardingState {
  switch (event.type) {
    case "RESET":
      return {
        step: "start",
        ctx: { draft: {}, intent: {}, missingFields: [] },
      };

    case "SET_INTENT": {
      const nextDraft = {
        ...state.ctx.draft,
        ...(event.primaryGoal ? { primaryGoal: event.primaryGoal } : {}),
        ...(event.timeHorizonYears ? { timeHorizonYears: event.timeHorizonYears } : {}),
      };
      const missingFields = computeMissingFields(nextDraft);
      return {
        step: state.step === "start" ? "intent" : state.step,
        ctx: { ...state.ctx, draft: nextDraft, intent: { ...state.ctx.intent, ...event }, missingFields },
      };
    }

    case "EXTRACTED_FIELDS": {
      const nextDraft = { ...state.ctx.draft, ...event.fields };
      const missingFields = computeMissingFields(nextDraft);
      return {
        step: state.step,
        ctx: { ...state.ctx, draft: nextDraft, missingFields, extractionConfidence: event.confidence },
      };
    }

      const nextDraft = { ...state.ctx.draft, ...event.confirmed };
      const missingFields = computeMissingFields(nextDraft);
      return { ...state, ctx: { ...state.ctx, draft: nextDraft, missingFields } };

    case "BASELINE_READY":
      return { ...state, step: "baseline_generated" };

    default:

write("src/onboarding/onboardingControllerImpl.ts", textwrap.dedent("""\
import type { OnboardingState } from "./onboardingState";
import type { OnboardingControllerDeps, ControllerResponse } from "./onboardingController";
import { onboardingReducer } from "./onboardingReducer";
import { computeMissingFields } from "./missingFields";
import { chooseNextQuestion } from "./nextQuestion";
import { uiHintFor } from "./questionUiHints";
import { finalizeFinancialInput } from "./finalizeFinancialInput";
import { renderExplanation } from "../renderers/atlasStrategyExplanationRenderer";
import { renderTracePanel } from "../renderers/tracePanelRenderer";

export class OnboardingController {
  constructor(private deps: OnboardingControllerDeps) {}

  async handleUserMessage(state: OnboardingState, userText: string, locale: string): Promise<ControllerResponse> {
    const extraction = await this.deps.extractionClient.extract(userText, {
      locale,
      prior: state.ctx?.draft ?? {},
    });

    let nextState = onboardingReducer(state, {
      type: "EXTRACTED_FIELDS",
      fields: extraction.fields,
      confidence: extraction.confidence,
    });

    if (extraction.intent?.primaryGoal || extraction.intent?.timeHorizonYears || extraction.intent?.biggestConcern) {
      nextState = onboardingReducer(nextState, {
        type: "SET_INTENT",
        primaryGoal: extraction.intent.primaryGoal,
        timeHorizonYears: extraction.intent.timeHorizonYears,
        biggestConcern: extraction.intent.biggestConcern,
      });
    }

    const missing = computeMissingFields(nextState.ctx.draft);
    nextState = { ...nextState, ctx: { ...nextState.ctx, missingFields: missing } };

    if (missing.length > 0) {
      const qKey = chooseNextQuestion(nextState.ctx);
      const qText = (this.deps.templateStore.get(`q.${qKey}`) ?? "A quick question to sharpen this…");
      const ui = uiHintFor(qKey);

      return {
        state: nextState,
        messages: [
          { role: "user", body: userText },
          { role: "assistant", tone: "calm", body: qText }
        ],
        expectedInput: { questionKey: qKey, ...ui }
      };
    }

    const finalized = finalizeFinancialInput(nextState.ctx.draft);
    const { config, meta } = await this.deps.configProvider.getStrategyConfig();
    const baseline = await this.deps.engineClient.runBaseline(finalized, config, meta);

    const renderedExplanation = renderExplanation(baseline.explainability, this.deps.templateStore);
    const panel = renderTracePanel(
      baseline.explainability,
      this.deps.templateStore,
      baseline.explainability.decisionTrace
    );

    const msgs = renderedExplanation.messages.map((m, i) => ({
      role: "assistant" as const,
      tone: m.tone,
      title: m.title,
      body: m.body,
      ctas: m.ctas,
      footnote: m.footnote,
      expandablePanel: i === 1 ? panel : undefined
    }));

    return {
      state: { ...nextState, step: "baseline_generated" },
      messages: [{ role: "user", body: userText }, ...msgs],
      baseline,
      renderedExplanation
    };
  }
}
"""))
